# AI CV Generator with Hono LangGraph BullMQ

This Python script demonstrates a simplified AI-powered LaTeX CV generator using LangGraph. It simulates the core AI steps of generating sections, formatting to LaTeX, and refining the output, mimicking the functionality described for TexFolio.

## Language

`python`

## How to Run

1. Ensure you have Python installed.
2. Install LangGraph: `pip install langgraph`
3. Run the script: `python texfolio_demo.py`
4. Follow the prompts to enter your skill.
